# Punch Jump! 🐒

Punch Jump! is a modern, web-based rendition of the classic Doodle Jump game, featuring the character Punch-kun. This project focuses on delivering a smooth and engaging gameplay experience with refined physics, precise hitboxes, and dynamic difficulty scaling. It also integrates with the Play.fun SDK for score submission and leaderboards.

## Features

*   **Doodle Jump-style Gameplay**: Experience the familiar vertical platforming challenge.
*   **Punch-kun Character**: Play as the charming Punch-kun.
*   **Refined Physics**: Enjoy proper physics implementation for realistic jumps and falls.
*   **Tight Hitboxes**: Precise collision detection for accurate gameplay.
*   **Difficulty Scaling**: The game adapts its challenge as your score increases.
*   **DeltaTime-based Game Loop**: Ensures consistent gameplay across different frame rates.
*   **Background Music & Volume Control**: Immerse yourself with background music and adjust volume to your preference.
*   **Responsive Controls**: Supports both keyboard (Arrow Keys/A/D) and touch controls for a seamless experience on desktop and mobile devices.
*   **Score Tracking**: Keep track of your current and high scores.
*   **Play.fun SDK Integration**: Submit your scores to the Play.fun platform for competitive play.

## Tech Stack

*   **HTML5**: For structuring the web page.
*   **CSS3**: For styling and visual presentation.
*   **JavaScript**: Powers the game logic, physics, and interactions.
*   **Canvas API**: Used for rendering game graphics.
*   **Play.fun SDK**: For game integration and score management.

## Installation & Setup

This game is designed to run directly in a web browser. No complex installation or build process is required.

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/makimyys-afk/punch-jump-game.git
    ```

2.  **Navigate to the project directory**:

    ```bash
    cd punch-jump-game
    ```

## Usage

To play the game, simply open the `index.html` file in your preferred web browser.

```bash
open index.html
```

### Controls

*   **Move Left**: `ArrowLeft` or `A` key
*   **Move Right**: `ArrowRight` or `D` key
*   **Restart Game (Game Over)**: `Spacebar`
*   **Touch Controls**: Swipe left or right on the screen to move.
